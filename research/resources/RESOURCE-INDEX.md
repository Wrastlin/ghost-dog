# Native Resource Index

> **This is your source-of-truth for ACTUAL documents.**
> Every entry below is a real, downloadable PDF or live web page from the TPA itself.
> Our markdown analysis files in `/tpa-profiles/` and `/industry-standards/` are separate.
>
> **To download all PDFs at once**, run from repo root:
> ```bash
> ./research/resources/download-all.sh
> ```
> PDFs land in the subfolders below. Some require manual download (noted).
